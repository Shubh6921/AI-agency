## 4. MILESTONE PHASES

### Phase 1: Foundation & Homepage (v1 Core)
- Setup Next.js, TypeScript, Tailwind, Lenis, and GSAP boilerplate.
- Establish the monochrome design system variables.
- Build Navbar, footer CTA, and Enquiry Drawer panel.
- Implement Homepage hero section, stats bento grid, and logo marquee loops.

### Phase 2: Content Pages & Form Backend
- Build Services and About pages.
- Create Work portfolio grid and Dynamic Case Study detail subroutes.
- Integrate Supabase Client and write form API routes.
- Build Multi-step Enquiry Form and FAQ Accordions on the Contact page.

### Phase 3: Polish, Motion, & Launch
- Bind Lenis smooth scrolling and GSAP scroll reveals to all page segments.
- Add magnetic pull dynamics on CTAs and implement custom follow-cursor.
- Perform SEO tuning, site metadata configuration, and accessibility audits.
- Deploy staging builds to Vercel and execute launch checks.

### Future Ideas
- **Interactive 3D Hero:** Spline/R3F interactive AI particle scene.
- **Client Dashboard:** Admin panel to view and label incoming enquiries in real-time.
- **Enquiry Webhooks:** Connect Supabase submissions directly to n8n pipelines for instant email alerts.
